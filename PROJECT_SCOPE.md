# PROJECT_SCOPE v1.0 — Buff

## Goal
Build a modular AI-assisted trading system for crypto markets (Binance Futures, 1H timeframe) that is auditable,
risk-controlled, and explainable.

## Scope Lock
- Exchange: Binance Futures (perpetual)
- Timeframe: 1H
- Universe: Top 10 by 30D avg volume ∩ Top 20 market cap (updated monthly)
- Data horizon: 3–4 years for research/backtesting

## Non-Goals (Hard Exclusions)
- Price prediction / "AI forecast"
- LLM-based execution or parameter overrides
- Heavy RL / deep learning in v1.0
- Social-media sentiment trading in v1.0
