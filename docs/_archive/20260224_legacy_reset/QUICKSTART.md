# Quickstart

1. Install Docker Desktop.
2. Start Buff:
   - Windows: double-click `RUN_ME_WINDOWS.bat`
   - Mac/Linux: run `bash RUN_ME_MAC_LINUX.sh` from the repo root
3. Local Docker Compose storage default: run artifacts are written to repo-local `.runs_compose`.
4. Open `http://localhost:3000` (the startup script opens it automatically when ready).
5. In the UI:
   - Import your CSV data.
   - Choose a strategy and risk level.
   - Create a run.
   - Watch progress, inspect results, and export the report.

For full troubleshooting and developer workflows, see `docs/05_RUNBOOK_DEV_WORKFLOW.md`.
